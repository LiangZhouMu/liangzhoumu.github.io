发表于 2020-10-08 更新于 2020-10-29

安装基本环境

  1. 下载 Node.js 和 git 并安装。

  2. 更改 npm 源并安装 cnpm。
    npm i -g cnpm --registry=https://registry.npm.taobao.org

  3. 接下来利用 cnpm 安装 hexo
    cnpm install -g hexo-cli
    并用hexo -v验证安装情况。

  4. 建立一个文件夹，以blog为例。进入blog ，并用 hexo init 初始化。

基本操作

hexo -n <name> 新建文件
hexo s 启动服务器。默认情况下，访问网址为：http://localhost:4000/
hexo generate 生成静态文件
hexo d 发布

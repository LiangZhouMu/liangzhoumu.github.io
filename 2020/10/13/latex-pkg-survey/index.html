TeX Live 中必要的宏包、文类与字体

发表于 2020-10-13 更新于 2020-10-29

目前 TeX Live 2020 的体积已经达到7GB之多，而当中大部分文件都是使用者闻所未闻的，包括众多的宏包、文类与字体。

但是，绝大部分的宏包都是普通用户不需要的、不必了解的，所以才有了这个备忘录用以记录必要的、有趣的宏包、文类、文件，以及相关支持信息。记录中的名称均以 Tex Live 为准。

必要的文类和宏包

一些基本的、必要的宏包与文类。

文类

除了标准文类外，比较值得推荐的就是memoir文类。

memoir 是一个特别优秀的文档类，重新实现了许多功能，隐藏了一些排版细节。

宏包

由于CTAN 中宏包可以根据自身依赖关系自动安装其他宏包，所以部分底层的宏包会被直接按照，因此不再出现在这一部分中。

基础宏包

必要的基本宏包、文类，包括 latex tools latex-bin.win32 latex-bin l3kernel l3packages l3experimental ctex 等。

字体

psnfss 文件名为 psfonts ，包括多款各类字体，如 Palatino，bookman， Helvetica，times 等，属于较为老牌的字体。而 palatino bookman helvetic mathpazo等宏包则会调用它们。

fontaxes

lm lm-math 即 latin-modern 字体。

newtx newpx 目前较为完善的字体宏包，符号较为齐全，对主流引擎支持较好。

libertinus libertinus-type1 libertinus-otf ibertinus-fonts 是一个著名的字体族，字体较全。

tex-gyre tex-gyre-math 是Psnfss 的 unicode 版，具有最丰富的 TrueType 数学字体支持。

ebgaramond ebgaramond-maths garamond-math 一款很漂亮的字体，数学支持可以借助ebgaramond-maths 宏包和Garamond-Math.otf 字体。同类型的garamond-libre字体也不错。

junicode 排印古英语

alegreya 排印小说

cm-unicode 基本字体的 unicode 版。

其他字体

除以上外，TeX Live 中还包含大量其他优质的西文字体以及少量其他语种字体，比如中文和日文。字体的格式主要分为两种，一种是 Type 1 字体，主要在 pdfTeX 引擎下使用，另一种是 Truetype 与 Opentype 字体，主要在 XeTeX、LuaTeX 等引擎下配合 fontspec 包使用。而且相当部分的字体同时提供多种格式以供选择。
